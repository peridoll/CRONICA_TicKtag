#### Status-Top System 開発
ステータス下限上限の固定用システムの開発

  各ステータス値の上限/下限を宣言させることで、
  そこまでしか上がらない(それ以下になる) → 移動不可など Speed上限を -4にする
  そこまでしか下がらない(それ以上になる) → 無敵状態など Speed下限を  3にする
  を実現する
